# 소수를 구하는 방법

1. 2부터 x의 제곱근 까지 나눠지는 수가 있는지 확인한다

```cpp
#include <cmath> //sqrt

bool isPrimeNumber(int num) {
  //1이하의 수는 소수가 아님
  if ( num <= 1 ) { 
    return false;
  }
  //2는 소수
  else if ( num == 2 ) { 
    return true;
  }

  int sqrt_num = sqrt(num);
  for ( int i = 2 ; i <= sqrt_num ; i++ ) { 
    if ( num % i == 0 ) { 
      return false;
    }   
  }
  return true;
}
```
